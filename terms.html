การใช้ Rundub แสดงว่าคุณยอมรับข้อกำหนดนี้ ใจความสำคัญ: ใช้อย่างเหมาะสม (ไม่ละเมิดผู้อื่น) · ดูแลสุขภาพตัวเอง (แอปไม่ใช่ที่ปรึกษาแพทย์) · GPS อาจคลาดเคลื่อน · เราอาจระงับบัญชีหากผิดกฎ

1. การยอมรับข้อกำหนด

เมื่อคุณติดตั้ง ลงทะเบียน หรือใช้แอป Rundub ("บริการ") คุณตกลงยอมรับข้อกำหนดเหล่านี้และ นโยบายความเป็นส่วนตัว หากคุณไม่ยอมรับ โปรดอย่าใช้บริการ

2. รายละเอียดบริการ

Rundub เป็นแอปชุมชนนักวิ่ง ครอบคลุม:

  • บันทึกการวิ่งด้วย GPS
  • ดู leaderboard ของสวนสาธารณะในประเทศไทย
  • Social features (follow, comment, mention, like)
  • กลุ่ม + challenges ระหว่างเพื่อน
  • Live tracking ขณะวิ่ง
  • นำเข้าข้อมูลจาก Strava / Health Connect
  • สร้างการ์ดแชร์การวิ่งสำหรับ social media

เราอาจเพิ่ม / ปรับปรุง / ลด ฟีเจอร์ได้ตลอดเวลา

3. คุณสมบัติผู้ใช้

  • อายุ 13 ปีขึ้นไป (ผู้ที่อายุต่ำกว่า 18 ปี ควรได้รับความยินยอมจากผู้ปกครอง)
  • สามารถใช้งานอินเทอร์เน็ตและอุปกรณ์มือถือได้
  • มีสุขภาพเหมาะสมกับการออกกำลังกาย (ดูข้อ 7)

4. ความรับผิดชอบของบัญชี

  • เก็บข้อมูล login (เบอร์โทร + OTP) เป็นความลับ
  • คุณรับผิดชอบทุกกิจกรรมในบัญชีของคุณ
  • แจ้งเราทันทีถ้าสงสัยว่ามีคนเข้าใช้บัญชีโดยไม่ได้รับอนุญาต
  • หนึ่งคน = หนึ่งบัญชี (ห้ามขายหรือโอน)

5. การใช้งานที่ยอมรับได้ / ห้ามทำ

ห้าม:

  • โพสต์ข้อมูลเท็จ (เช่น run ที่ไม่ได้วิ่งจริง / ใช้รถ / โกง GPS)
  • คุกคาม ข่มขู่ ดูถูก หรือใช้คำหยาบคายต่อผู้อื่น
  • โพสต์เนื้อหาที่ผิดกฎหมาย ผิดศีลธรรม หรือลามก
  • เผยแพร่ข้อมูลส่วนตัวของผู้อื่นโดยไม่ได้รับอนุญาต
  • ใช้ bot, scraper, หรือเครื่องมืออัตโนมัติเพื่อ access บริการ
  • พยายาม reverse engineer, decompile, หรือเจาะระบบ
  • ใช้ Live tracking เพื่อ stalking / harassment
  • โฆษณา / ขายของในคอมเมนต์โดยไม่ได้รับอนุญาต

การละเมิดอาจส่งผลให้ลบ content, ระงับ, หรือลบบัญชีถาวร (ดูข้อ 11)

6. เนื้อหาที่ผู้ใช้สร้าง (UGC)

  • กรรมสิทธิ์ — คุณยังเป็นเจ้าของ run / comment / รูป ที่โพสต์
  • สิทธิ์ที่ให้เรา — คุณให้สิทธิ์เราในการ host, แสดง, แชร์, และส่งต่อให้ผู้ใช้คนอื่นตาม privacy setting ที่คุณตั้ง — เพื่อให้บริการทำงาน
  • เราอาจลบ — content ที่ละเมิดข้อกำหนด หรือกฎหมาย โดยไม่แจ้งล่วงหน้า

7. คำเตือนด้านสุขภาพ

Rundub ไม่ใช่ที่ปรึกษาทางการแพทย์

การวิ่งและออกกำลังกายมีความเสี่ยงต่อสุขภาพ ก่อนเริ่มโปรแกรมการออกกำลังกายใหม่ ควรปรึกษาแพทย์ โดยเฉพาะถ้าคุณ:

  • มีโรคหัวใจ ความดัน หรือเบาหวาน
  • เคยบาดเจ็บกระดูก ข้อ หรือกล้ามเนื้อ
  • ตั้งครรภ์หรือหลังคลอดใหม่
  • อายุ 60 ปีขึ้นไป และไม่เคยออกกำลังกายเป็นประจำ

ข้อมูลและสถิติในแอปไม่ใช่คำแนะนำทางการแพทย์ การใช้แอปนี้คุณยอมรับความเสี่ยงของการออกกำลังกายเอง

8. ความถูกต้องของ GPS และข้อมูล

  • GPS บนสมาร์ทโฟนอาจคลาดเคลื่อน — ระยะทาง, เพซ, และเส้นทางที่บันทึกอาจไม่แม่นยำ 100%
  • ข้อมูลจาก Strava / Health Connect / นาฬิกาออกกำลังกาย ขึ้นกับแหล่งข้อมูลนั้นๆ — เราเป็นแค่ผู้แสดง
  • ไม่ควรใช้แอปเป็นเครื่องมือนำทาง — เราไม่รับผิดถ้าหลงทาง / บาดเจ็บ
  • Live tracking ขึ้นกับสัญญาณมือถือ — อาจดีเลย์หรือขาดหาย
